fennix
======

El objetivo de este proyecto es el de crear material de calidad sobre la seguridad informática,  programación, etc. 

Hoy en día cada foro y comunidad de seguridad crea su propio material, muchas veces ese material no esta completo del todo, es muy básico, o tiene abundantes faltas ortográficas. Por eso en este proyecto decidimos usar github para que cualquier usuario de cualquier lugar, no solo de Hack x Crack (foro de donde salio este proyecto originalmente) pueda proponer cambios o correcciones al material, consiguiendo de esta forma contenido de mayor calidad.
